Entity Operation URL Method Profile Description
Issue get localhost:8082/issue/id GET Admin Return the issue from the id

Issue get localhost:8082/issue/id GET Developer Return only the issues from the project that the developer belong
Issue count localhost:8082/issue/id GET Admin An admin gets the number of all the issues
